# MO - Multimodal Observer
## How to build
- In netbeans -> easy
- Command line -> not so easy

### Prerequesites
- JDK 1.8

#### Netbeans
1. Install Netbeans 8
2. Add Gradle Plugin to Netbeans
3. Download sources and open the project in Netbeans
4. Press Build
